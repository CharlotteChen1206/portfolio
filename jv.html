Japanese Village – Restaurant Website Redesign

Firebase database integration, Email notifications (EmailJS), JSON content rendering, Front-end web development, Agile collaboration in sprint format

Introduction

This 9-week capstone project involved creating a new logo and redesigning the website for Japanese Village, a restaurant based in Calgary. I worked as the web developer, collaborating closely with a UX designer and a graphic designer across three sprints. My task was to build a modern, functional website that stays true to the restaurant’s existing style. I rebuilt the site structure, redesigned the menu page using JSON for easier content management, and developed a reservation system using Firebase and JavaScript. To improve customer interaction, I also implemented email notifications using EmailJS.

Research & Data

We reviewed the existing website to identify areas for improvement and conducted design alignment checks to ensure consistency with the brand. User flow and content structure were enhanced for a better browsing and booking experience.
