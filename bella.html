Bella Cosmetic – Makeup Brand Website with API Integration

Logo design & color branding, API integration (Makeup API), Basic front-end development (HTML, CSS, JavaScript), Dynamic product gallery, Dropdown filter

Introduction

Bella Cosmetic is an early project where we created a simple brand website for a fictional makeup line targeting the young generation. I designed the logo and chose a soft pink color palette to reflect a youthful, feminine identity. The goal was to present products in an appealing, easy-to-navigate layout. On the development side, I integrated a Makeup API to dynamically display product data in a photo gallery, and implemented a dropdown filter to let users browse by category or brand.

Research & Data

We looked into existing cosmetic brand websites to understand how they showcase products, use filters, and structure product pages. This helped shape our layout and user flow, especially focusing on simplicity and visual appeal for younger audiences.
