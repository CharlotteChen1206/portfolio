FitLife – Fitness Website with YouTube API & Subscription Checkout

Branding & logo design, Collaboration in web design & dev teams, API integration (YouTube), UX research & content planning

Introduction

FitLife is a team project focused on building a fitness and gym website that integrates real-time workout content and a functional subscription checkout system. I led the creation of the logo and brand guideline, ensuring the visual identity aligned with a modern, energetic fitness brand. I collaborated closely with both the web designer and developer to keep the design cohesive across pages. The project centered on two technical goals: embedding workout tutorials using the YouTube API, and building a plan-based checkout to allow users to subscribe to different fitness packages.

Research & Data

We reviewed popular fitness websites to analyze what features users expect, such as video content, trainer profiles, pricing tiers, and easy navigation. This guided our structure and helped identify where API integration and payment features would best enhance user experience.
